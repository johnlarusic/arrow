arrow by johnlarusic

Bottleneck traveling salesman problem solver (BTSP), plus solvers for related problems.

Still very much under development, but should be more or less together by Fall 2009 (my thesis involves experimental study using this code :-).

Dependencies

Git v1.6.1: http://git-scm.com/ Python v2.6.1: http://www.python.org/ Scons v1.2.0: http://www.scons.org/ CMPH v0.8: http://cmph.sourceforge.net/ Concorde 03.12.19: http://www.tsp.gatech.edu/concorde/
